# 🚗 Car Price Prediction

## 📌 Project Overview

This project focuses on predicting car prices using machine learning techniques. The model analyzes various features such as car age, mileage, fuel type, and transmission to estimate the selling price of a car.

## 🎯 Objective

To build a regression model that can accurately predict the price of a car based on its features.

## ⚙️ Tech Stack

* Python 🐍
* Pandas & NumPy
* Scikit-learn
* Matplotlib & Seaborn
* Jupyter Notebook

## 📊 Dataset Features

* Year of manufacture
* Present price
* Kilometers driven
* Fuel type
* Seller type
